# Mandelbrot Set

*Max 8.3*

This Max Patch creates the Mandelbrot Set using jit.gl.pix and codebox.

The controls are:<br/>
**iter**: number of iterations<br/>
**zoom**<br/>
**xPos**: x-position<br/>
**yPos**: y-position<br/>
**shape**: changes the shape<br/>
**color**: changes the colors<br/>
